# INF492-CV-TF

Repositório para o desenvolvimento do Trabalho Final da disciplina INF492 - Fundamentos de Visão Computacional.

**Alunos:** José Roberto e Thiago Zimerer

**Tema:** Histogram of Oriented Gradients (HOG)

## Instruções gerais:

1. O arquivo 'environment.yml' contém as dependências que precisam ser instaladas para viabilizar a execução dos códigos.

2. A execução dos códigos é bem simples, bastando executar as células de cada um dos notebooks. Especificamente para o notebook 'hog.ipynb' é necessário, caso haja interesse de executar os códigos para outras imagens, definir o path da imagem de entrada na segunda célula de código Python.
